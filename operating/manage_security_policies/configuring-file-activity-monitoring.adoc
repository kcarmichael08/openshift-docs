= Configuring file activity monitoring
:context: file-activity-monitoring
:toc: macro
:toclevels: 3

toc::[]

[role="_abstract"]
Configure policies in Red{nbsp}Hat Advanced Cluster Security (RHACS) to detect and alert on sensitive file accesses or operations, for containerized workloads or cluster nodes. Define criteria to identify high risk accesses of sensitive files, such as opening a file for writing, file deletion or permission changes, and optionally trigger notifications. Set enforcement actions to automatically block noncompliant workloads.
