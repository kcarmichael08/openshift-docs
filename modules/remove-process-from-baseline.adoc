= Removing a process from the baseline

[role="_abstract"]
You can remove a process from the baseline so that it is not considered when the system evaluates the baseline.

.Procedure
. In the {product-title} ({product-title-short}) portal, select *Risk* from the navigation menu.
. Select a deployment from the list of deployments in the default *Risk* view.
. Select the *Process discovery* tab.
. Under the *Spec Container baselines* section,  click the *Remove* icon for the process you want to remove from the process baseline.
