= Creating a security policy from the system policies view

[role="_abstract"]
Define a new security policy in the {product-title-short} portal to enforce security rules in your environment. Go to *Policy Management* to create a new policy and enter the required definition information. You can manage security settings by creating policies in the *Platform Configuration -> Policy* page.

.Procedure
. In the {product-title-short} portal, go to *Platform Configuration* -> *Policy Management*.
. Click *Create policy*.
. Configure the policy definition information in the following sections.
