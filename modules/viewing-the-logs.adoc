= Viewing the logs

[discrete]
== Kubernetes

.Procedure

* Run the following command to view the logs for the Central pod:
+
[source,terminal,subs="+quotes"]
----
$ kubectl logs -n stackrox _<central_pod>_
----

[discrete]
== {ocp}

.Procedure

* Run the following command to view the logs for the Central pod:
+
[source,terminal,subs="+quotes"]
----
$ oc logs -n stackrox _<central_pod>_
